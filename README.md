# Route Vision

India route planner: live OpenStreetMap driving directions plus teaching-hub graph algorithms (Dijkstra, Bellman–Ford, Floyd–Warshall, TSP) with comparison charts.

## Scripts

- `npm run dev` — local dev server
- `npm run build` — production build
- `npm run test` — Vitest unit tests
- `npx playwright test` — E2E (requires dev server or uses `webServer` in config)
